Leitura de Tipos de Dados em C#
Este programa em C# demonstra a leitura de diferentes tipos de dados a partir da entrada do usuário, como inteiros, números decimais (float), caracteres e booleanos.

Funcionalidades
Entrada de Dados: O programa solicita ao usuário que insira um valor para cada tipo de dado.

Conversão de Tipos: Utiliza os métodos int.Parse(), float.Parse(), char.Parse() e bool.Parse() para converter a entrada de texto (string) para o tipo de dado correspondente.

Exibição de Dados: Imprime todos os valores inseridos na tela, confirmando a conversão e a leitura corretas.
